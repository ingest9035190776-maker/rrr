# 👨‍👩‍👧‍👦 Семейный помощник

Телеграм-бот для семьи: дневник кормлений, список дел, ИИ-общение, игры.

## Возможности

- 🍼 Дневник кормлений (объем, время, комментарии)
- 📊 График и статистика кормлений
- 📋 Общий список дел
- 🎭 Режимы общения (Семейный, Веселый, Деловой, Детский)
- 🎮 Игры: загадки, шутки, комплименты
- 🤖 ИИ-ответы в вашем стиле

## Установка

1. Клонируйте репозиторий
2. Создайте файл `.env` с токеном
3. Установите зависимости: `pip install -r requirements.txt`
4. Запустите: `python main.py`

## Деплой на Render

1. Подключите GitHub-репозиторий
2. Build Command: `pip install -r requirements.txt`
3. Start Command: `python main.py`
4. Добавьте переменную `TELEGRAM_TOKEN`
